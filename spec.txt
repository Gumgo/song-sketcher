A utility to quickly record and arrange clips into a song

- Left vertical menu bar
	- New
	- Save
	- Save as
	- Load
	- Settings (for mic/speakers)

- Right vertical menu bar (only visible when a project is loaded)
	- Top
		- Play/pause
		- Stop
		- Metronome enable/disable
	- Bottom
		- Undo
		- Redo

- Main screen
	- Top half - timeline
		- Track listing on the left - mute/solo option
		- Can vertically scroll to see more tracks
		- Horizontal scrollable measure grid to the right of track listing
		- Actions:
			- Add clip: click an empty measure and the selected clip is inserted (if there is enough space)
			- Remove clip: click the existing clip in a measure
			- Selecting/moving clips: TODO
	- Bottom half - library
		- Clip categories on the lift
		- Can vertically scroll to see more categories
		- Horizontal scrollable clip grid to the right of clip categories
		- Actions:
			- Add category: click the [+] button below the last category
			- Removing category: right click and select "remove"
			- Rename/recolor category: right click and select "rename" (popup box is displayed)
			- Add clip: click the [rec] button at the end of a clip category
			- Remove clip: right click clip and select "remove"
			- Edit clip: double click clip
			- Move clip: click and drag
			- Select clip: click on it

- Clip editor
	- Name entry box
	- Waveform display
		- Shows intro and outro
		- Can trim intro/outro
	- Bottom horizontal bar:
		- Left side
			- Record
			- Play/pause
			- Stop
			- Metronome enable/disable
		- Right side
			- Reject button
			- Accept button

- New project dialog
	- Name
	- Sample rate
	- Beats per minute
	- Beats per measure
